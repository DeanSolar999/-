---成績計算方式---

1. 各隊於循環賽中的總得計算之，總分最高為冠軍，其次為亞軍&季軍。
2. 如遇兩隊總分相同時，則沿用兩隊競賽時之勝負。
3. 如遇三隊之勝分相同的情況，則比較失分，失分較少者為勝。
